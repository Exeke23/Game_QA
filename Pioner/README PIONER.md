### #1: Дюп патрон для дробовика "Пробудитель"
- **Severity:** Critical
- **Priority:** ASAP
- **Описание:** При перекладывании оружия из инвентаря в руку патроны могу дюпнуться
- **Предусловия:** Иметь один незаряженный патрон для дробовика в инвентаре 
- **Шаги воспроизведения:**
	1. Перезарядить оружие с помощью ЛКМ
	2. Выстрелить из дробовика
	3. Открыть инвентарь с помощью радиального меню и сложить дробовик драг енд дропом в инвентарь
	4. Переместить дробовик в руку с помощью драг енд дропа в руку и закрыть инвентарь 
	5. Перезарядить дробовик с помощью появившегося патрона
- **Ожидаемый результат:** Патроны для дробовика не дюпаются после перемещения дробовика из слота руки в инвентарь и обратно
- **Фактический результат:** Патроны для дробовика дюпаются после перемещения дробовика из слота руки в инвентарь и обратно
- **Видео:** [ссылка](https://disk.yandex.ru/i/jrZn91G_ZhC-Kw)
- **Комментарий:** Ну еще можно докопаться до того, что персонаж заряжает сразу два патрона вместо одного доступного, но это наверно игровая условность(хотя если перезарядить только один патрон из двух доступных в стволе, то персонаж перезаряжает только один патрон)
### #2: Нет возможности перемещаться по инвентарю с помощью геймпада
- **Severity:** Critical
- **Priority:** High
- **Описание:** В инвентаре не работает выбор категорий предметов с помощью правого стика, также нельзя выбрать определенный предмет в инвентаре с помощью геймпада
- **Шаги воспроизведения:**
	1. Открыть инвентарь
	2. Выбрать активную область со списком предметов с помощью кнопки A
	3. Выбрать категории предметов с помощью правого стика
- **Ожидаемый результат:** Категории предметов выбираются, можно спуститься в сам инвентарь и выбрать определенный предмет
- **Фактический результат:** Категории предметов не выбираются, нет возможности можно спуститься в сам инвентарь и выбрать определенный предмет
- **Видео:** [ссылка](https://disk.yandex.ru/i/lgVXNI7JDwzqdQ)
### #3: Оружие не атакует выбрав его через меню быстрого доступа играя на геймпаде 
- **Severity:** Critical
- **Priority:** High
- **Описание:** Выбрав оружие через меню быстрого доступа открываемое кнопкой RS или правый стик оружие не атакует
- **Шаги воспроизведения:**
	1. Открыть инвентарь 
	2. Экипировать оружие в один из слотов для оружия и закрыть инвентарь
	3. Нажать на правый стик и выбрать оружие из меню
	4. Нажать на кнопку RT для атаки 
- **Ожидаемый результат:** Персонаж атакуют/стреляет с оружия
- **Фактический результат:** Персонаж не атакуют/стреляет с оружия, оружия убирается и достается данной кнопкой, при этом можно поймать тайминг быстро нажать повторно на кнопку атаки. При этом воспроизводится не характерный звук для оружия
- **Видео:** [ссылка](https://disk.yandex.ru/i/fHGjRmzmUYR0lA)
### #4: Отсутствует коллизия у двух объектов
- **Severity:** Major
- **Priority:** Medium
- **Описание:** На стартовой локации отсутствует боксовая коллизия у объекта Сейф и Умывальник, игрок проходит сквозь них
- **Шаги воспроизведения:**
	1. После загрузки новой игры закончить диалог с персонажем Эдгар
	2. Пройти сквозь объект Сейф
	3. Пройти сквозь объект Умывальник
- **Ожидаемый результат:** Коллизия у двух объектов присутствует, игрок не проходит сквозь объекты
- **Фактический результат:** Коллизия у двух объектов отсутствует, игрок свободно проходит сквозь объекты
- **Видео:** [ссылка](https://disk.yandex.ru/i/mE1Wuh5Kw202UQ)
### #5: Некорректная иконка кнопки в разделе Прогресс
- **Severity:** Minor
- **Priority:** Low
- **Описание:** В меню паузы при неподключенном геймпаде в разделе Прогресс над наименованием ветки прокачки отображается некорректная иконка кнопки взаимодействия 
- **Шаги воспроизведения:**
	1. Войти в игру через главное меню игры
	2. После загрузки нажать на кнопку Escape 
	3. Выбрать раздел Прогресс
- **Ожидаемый результат:** Отображается клавиатурная иконка кнопки для взаимодействия с веткой прокачки
- **Фактический результат:** Отображается геймпадная иконка кнопки для взаимодействия с веткой прокачки 
- **Скрин:** [ссылка](https://disk.yandex.ru/i/6S8TrYLd_k1peg)

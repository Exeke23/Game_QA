#### Обо мне
Меня зовут Андрей, я инженер по тестированию игр. Выбрал данную профессию так как очень люблю видеоигры и хочу помогать делать игры более качественными для игроков.
#### О моем опыте
Есть 2х летний опыт в тестировании. Тестировал в основном сайты, а именно интернет-магазины, также участвовал в тестировании мобильного приложения где тестировал REST API с помощью инструмента Postman.
#### Навыки
- Уверенное использование теории и техник тест-дизайн в работе 
- Создание и поддержка тест-кейсов и чек-листов
- Работа с баг-трекерами (Test IT, DoQa, Jira, TestRail)
- Основы работы с Unity и Unreal Engine
